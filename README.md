# SUI BULK SEND (Youchaole) 🚀

一个复古风格 (Retro-Pixel) 的 Sui 链上代币批量分发工具。

## ✨ 特性

- **多网络支持**：完美支持 Sui Mainnet 和 Testnet。
- **全代币适配**：自动识别 SUI 原生币及所有自定义代币 (WAL, USDT 等) 的余额与精度。
- **智能过滤**：自动去重地址，并自动排除当前发送者地址以节省 Gas 费。
- **极致体验**：支持分发完成后自动定位交易结果，双浏览器 (Suiscan/Suivision) 快捷查看。
- **复古 UI**：采用 Press Start 2P 字体与像素级 UI 设计。

## 🛠️ 技术栈

- **Move**: Sui Smart Contract
- **Frontend**: Vite + React + TypeScript
- **Styling**: Tailwind CSS + Pixel Art Theme
- **SDK**: @mysten/dapp-kit
